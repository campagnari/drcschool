# `drcschool` --- DRC' s LaTeX school class

## Beschreibung
LaTeX Klasse zur herstellung von Unterrichtsentwürfen, Arbeitsblättern und Klassenarbeiten in den Fächern Mathematik und Physik in der Schule.
